📋 Grading Criteria

Score each criterion from 0 to the maximum points shown. Total possible: 100 points. Minimum passing score: 80 points (80%).

Criterion Requirements Max Points Score
1. Document Structure
  • Correct DOCTYPE declaration on all pages
  • Proper html, head, body structure
  • UTF-8 charset and viewport meta tags
  • Valid HTML5 (passes W3C validator)
 10
2. Semantic HTML5
  • Uses header, nav, main, article, section, aside, footer appropriately
  • One h1 per page with proper heading hierarchy
  • No skipped heading levels (h1→h2→h3, not h1→h3)
  • Minimal use of generic div/span for structure
 15
3. Navigation & Links
  • Consistent navigation across all 5 pages
  • All internal links work correctly
  • External links use target="_blank" + rel="noopener noreferrer"
  • At least one internal anchor link (#section)
  • Email link using mailto:
 10
4. Content Elements
  • At least one accessible table (caption, thead, tbody, scope)
  • At least one form with 5+ fields and validation
  • Both ordered and unordered lists used
  • At least 4 images with figure/figcaption
  • Text emphasis tags used (strong, em, mark)
 15
5. Accessibility
  • All images have descriptive alt text
  • All form inputs have associated labels
  • Language attribute set on html element
  • Tables use scope attributes
  • Semantic structure supports screen readers
 15
6. SEO & Metadata
  • Unique, optimized title on every page (<60 chars)
  • Meta description on every page (150-160 chars)
  • Open Graph tags on homepage
  • Favicon linked
  • Canonical link present
 10
7. Code Quality
  • Consistent indentation (2 or 4 spaces)
  • Logical file organization (folders for css, images, etc.)
  • Helpful HTML comments for major sections
  • No inline styles
  • Clean, readable code
 10
8. Project Completeness
  • All 5 required pages present (Home, About, Menu/Services, Gallery, Contact)
  • Each page has substantial, meaningful content
  • Project feels complete and professional
  • Creative effort and attention to detail
 15

📝 Instructor Notes

💡 Grading Tips:
  • Use the W3C Validator to check for HTML errors
  • Test all links by clicking through every page
  • Check accessibility with browser DevTools (Lighthouse audit)
  • View page source to check indentation and comments
  • Verify images load and have alt text by inspecting them
